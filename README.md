## 🏆 Flagship Projects

### 🧠 Zaforge
> Full-stack spatial planning tool — Notion meets Obsidian meets a node-graph canvas

A canvas/node-graph planning application with polymorphic node types and three synchronized views — canvas, table, and Gantt. Built backend-first: Supabase authentication (including Google OAuth) and PostgreSQL persistence shipped before any AI or visual polish, by design.

`React` `Supabase` `PostgreSQL` `Node-Graph UI`

---

### 🎮 AKIRA Studios
> Game studio landing page — Week 1 capstone, pure CSS, zero JavaScript libraries

A cinematic game-studio brand page featuring a text-behind-image hero, a full featured-game showcase, a bento-style game library, and a horizontally-scrolling character gallery — built entirely with hand-written CSS animations and scroll-driven reveals.

**[Live Demo →](https://60-days-frontend-challenge.vercel.app)**

`Pure CSS` `Scroll Animations` `CSS Grid` `Design Systems`

---

### 🖥️ Aura OS
> Browser-based desktop OS with a live IDE and an AI-powered backend

A fullscreen, browser-based operating system simulation — window manager, live code editor, terminal emulator, and a Python FastAPI backend wired to Gemini function-calling for intelligent automation.

`React` `Python` `FastAPI` `Gemini API` `Window Management`

---

### 🌐 NovaMQ
> A C++20 message broker built from first principles

A single-threaded, event-loop-based message broker built on Asio, with structured JSON serialization and passive disconnect detection — written and understood line by line, not generated.

`C++20` `Asio` `Systems Programming` `Networking`

---

### 🐺 WOLF Acoustics
> GLB product showcase with real-time material switching

A Three.js-powered 3D headphone showcase with real-time color/material switching, built as a deep dive into WebGL rendering pipelines and interactive 3D commerce.

`Three.js` `WebGL` `3D Product Rendering`

---

### 🕷️ Spider-Verse Cinematic Story
> A scroll-driven narrative experience, told in pure CSS

A full cinematic storytelling page — original character-driven copywriting, a scroll progress indicator, and section-by-section reveals timed to viewport position, not scroll duration.

`CSS Scroll-Timeline` `Storytelling UI` `Animation Choreography`
