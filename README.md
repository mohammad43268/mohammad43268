## 🏆 Flagship Projects

<table>
<tr>
<td width="50%" valign="top">

### 🧠 Zaforge
**Full-stack spatial planning tool** — Notion meets Obsidian meets a node-graph canvas.

A canvas/node-graph planning app with polymorphic node types and three synchronized views: canvas, table, and Gantt. Built backend-first by design — Supabase authentication (with Google OAuth) and PostgreSQL persistence shipped *before* any AI or visual polish.

`React` `Supabase` `PostgreSQL` `Node-Graph UI`

**Status:** Auth complete → UI polish & performance in progress → AI features next

</td>
<td width="50%" valign="top">

### 🎮 AKIRA Studios
**Game studio landing page** — Week 1 capstone, pure CSS, zero JS libraries.

A cinematic game-studio brand page: text-behind-image hero, featured-game showcase, bento-style game library, and a horizontally-scrolling character gallery — every animation hand-written in CSS.

`Pure CSS` `Scroll-Timeline` `CSS Grid`

**[▶ Live Demo](https://60-days-frontend-challenge.vercel.app)**

</td>
</tr>
<tr>
<td width="50%" valign="top">

### 🖥️ Aura OS
**Browser-based desktop OS** with a live IDE and an AI-powered backend.

A fullscreen OS simulation in the browser — window manager, live code editor, terminal emulator, and a Python FastAPI backend wired to Gemini function-calling for intelligent automation.

`React` `Python` `FastAPI` `Gemini API`

</td>
<td width="50%" valign="top">

### 🌐 NovaMQ
**A C++20 message broker**, built from first principles.

A single-threaded, event-loop-based broker on Asio with structured JSON serialization and passive disconnect detection — written and understood line by line, not generated.

`C++20` `Asio` `Systems Programming`

</td>
</tr>

### 🐺 WOLF Acoustics

`Three.js` `WebGL` `3D Rendering`

</td>
<td width="50%" valign="top">

### 🕷️ Spider-Verse Cinematic Story

Original character-driven copywriting, a scroll progress indicator, and section reveals timed to viewport position — not scroll duration.

`CSS Scroll-Timeline` `Storytelling UI`
